# Financial Product Scraper

Personal exercise to improve scraping data skill with Node JS.


### Dependencies

* Node js >= 8
* NPM    >= 6

## Getting Started

* git clone https://github.com/abdurhasan/finance-scraping
* cd finance-scraping
* npm install
* npm start
* Open in your browser : http://localhost:7000/
